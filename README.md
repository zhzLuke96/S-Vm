# README
    大多函数语言都是基于lambda设计的，lambda作为一种属性包含在了其中，也由此放弃了一定的可计算性
    全知全能的lambda要是作为最外层会如何呢？

    在这里我将用lambda去尝试实现类lisp的语法，实践留下无限的可计算性的意义

> 一点微小的工作
